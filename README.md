# Registration Form with Real-Time Validation

## Project Overview

This project is a simple Registration Form built using HTML, CSS, and JavaScript. It performs real-time validation for user inputs and enables the submit button only when all fields are valid.

## Features

* Name field validation (cannot be empty)
* Email format validation
* Password length validation (minimum 6 characters)
* Real-time error messages
* Disabled submit button until all validations pass
* Clean and responsive user interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Validation Rules

### Name

* Cannot be empty

### Email

* Must follow a valid email format
* Example: [user@example.com](mailto:user@example.com)

### Password

* Must contain at least 6 characters
